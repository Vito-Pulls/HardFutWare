# HardFutWare

HardFutWare es una página web personal que combina noticias de hardware y del FC Barcelona, creada completamente con HTML, CSS y JavaScript puro. La idea es demostrar un diseño coherente, adaptable y funcional, sin depender de frameworks externos ni servidores.

🌐 Demo (GitHub Pages): [https://vito-pulls.github.io/HardFutWare/](https://vito-pulls.github.io/HardFutWare/)  
📁 Repositorio: [https://github.com/Vito-Pulls/HardFutWare](https://github.com/Vito-Pulls/HardFutWare)

---

## 🚀 Características

- Interfaz limpia y semántica.
- Diseño adaptable (responsive) para móviles y tabletas.
- Cambio de tema dinámico entre "Blau" y "Grana".
- Navegación rápida entre secciones: Noticias, Hardware, Videos y Contacto.
- Formulario de contacto funcional.
- Todo el código incluido en un único documento HTML para fácil comprensión y modificación.

---

## 🛠️ Funcionalidades

✅ Secciones de contenido organizadas: Noticias, Hardware, Videos, Contacto.  
✅ Cambio de esquema de colores con un botón.  
✅ Navegación fija (sticky) para facilitar el acceso a cualquier sección.  
✅ Tarjetas interactivas para mostrar noticias y videos.  
✅ Formularios con validación básica de campos.  
✅ Estilo responsivo con CSS Grid y Flexbox.

---

## 💡 Cómo funciona

- Cada sección del sitio está construida con HTML semántico y contiene tarjetas con información relevante.
- Se emplean variables CSS para los colores y estilos globales.
- El botón de cambio de color alterna entre los temas “Blau” y “Grana”, manipulando las clases del `body` mediante JavaScript.
- El header se mantiene visible y cambia su apariencia al hacer scroll.

---

## 💻 Uso

1. Clona o descarga el repositorio.
2. Abre `index.html` en cualquier navegador moderno.
3. Navega por las secciones y prueba el botón de cambio de tema.
4. No requiere instalación ni servidor.
